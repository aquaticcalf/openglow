## my opencode setup

to clone this setup for your opencode use, run the following command in your terminal :

```bash
rm -rf ~/.config/opencode && git clone --depth 1 https://github.com/aquaticcalf/.opencode.git ~/.config/opencode/
```

opencode will automatically turn sexy in your next session!

### pre requisites

this setup assumes that you have the following programs installed :

- [opencode](https://opencode.ai)
- [git](https://git-scm.com)
- [ghostty](https://ghostty.org)
- [btca](https://btca.dev)
- [nvim](https://neovim.io)
